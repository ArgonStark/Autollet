# Autollet

AI-powered agent wallet using Chainlink CRE, Coinbase CDP Embedded Wallets, and LLM integration.

## Project Structure

- `/backend` - Express + TypeScript backend
- `/frontend` - Next.js + TypeScript frontend
- `/shared` - Shared interfaces and schemas

## Setup

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Shared
```bash
cd shared
npm install
npm run build
```
